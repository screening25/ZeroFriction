## [v0.4.4] — 2026-05-22 · Excel 및 PDF 내보내기 기능 추가

### ✨ 일정·재고·메모 데이터 추출 기능 추가

| 변경 전 (Before) | 변경 후 (After) |
| --- | --- |
| 데이터를 외부 엑셀 문서나 인쇄물로 출력할 수 있는 방법이 부재했음 | 각 탭 및 독립적인 뷰어 페이지 상단에 '엑셀' 및 'PDF' 버튼을 추가하여 다운로드 및 인쇄가 가능하도록 개선 |
| 엑셀로 내보낼 때 한글이 깨지는 현상이 발생할 가능성이 있었음 | Excel 한글 깨짐 방지를 위해 UTF-8 BOM(\uFEFF)을 자동으로 추가하여 신뢰도 높은 CSV를 다운로드하도록 구현 |
| PDF로 인쇄할 때 원본 레이아웃이 뭉개지거나 스타일이 유실됨 | 보이지 않는 임시 iframe을 활용해 현대적이고 깔끔하게 스타일링된 출력 전용 문서를 동적으로 생성하여 인쇄(PDF 저장) 기능 실행 |
| 메모 내용에 포함된 마크다운 서식이 텍스트로 보임 | 메모 PDF 인쇄 시에도 인라인 마크다운 서식, 표, 코드 블록, 리스트, 인용구, 태그 및 멘션 등이 완벽히 렌더링되도록 경량 마크다운 파서 연동 |
| 정렬 규칙 없이 등록된 순서대로 추출됨 | 일정은 날짜/시간 오름차순, 재고는 카테고리/품명 오름차순, 메모는 고정 상태/날짜 내림차순으로 사전 정렬하여 내보내도록 편의성 강화 |
| 메모 상세 조회 모달에서 개별 메모를 별도로 파일 추출하거나 인쇄할 수 없었음 | 메모 상세 보기(읽기 전용 모드) 헤더에 엑셀 및 PDF 내보내기 전용 단추를 배치하여 단일 메모를 개별적으로 CSV 다운로드하거나 PDF 인쇄가 가능하도록 개선 |
| 화면 크기가 작아질 때 상단의 엑셀/PDF 내보내기 버튼의 크기가 뭉개지고 작아짐 | 엑셀 내보내기 버튼의 라벨명을 영어 'Excel'로 변경하고, 모든 내보내기 버튼(Excel & PDF)에 축소 방지 스타일(flexShrink: 0)을 부여하여 화면 크기와 무관하게 일정한 레이아웃 크기를 유지하도록 고도화 |

## [v0.4.3] — 2026-05-22 · 일정 옵션 개선 및 메모 마크다운 확장

### ✨ 일정 '하루 종일' (All Day) 옵션 지원

| 변경 전 (Before) | 변경 후 (After) |
| --- | --- |
| 모든 일정에 특정 시간을 필수 지정해야 하여 하루 종일 이어지는 이벤트를 표현하기 어려움 | 일정 등록/수정 모달에 '하루 종일' 토글 스위치 제공. 활성화 시 시간 선택 창이 숨겨지며 '하루 종일' 배지로 대체됨 |
| 일정 리스트에서 시간순 정렬만 지원하여 시간 없는 일정이 뒤섞임 | 하루 종일 일정(`attrs.allDay === true`)을 당일 리스트의 가장 최상단에 우선 정렬 |
| 대시보드 및 일정 목록에 무조건 시간 텍스트만 표시됨 | 대시보드 리스트와 달력 일일 목록에 각각 '하루 종일' 또는 '종일' 배지 노출 |
| 백업 원본 데이터나 CSV 파일에 시간만 표시되거나 알림 체크 대상에 포함됨 | CSV 내보내기 시 '하루 종일'로 출력되며, 백업 탐색기에도 명시. 백그라운드 알림 발생 조건에서 자동 배제 |

### ♻️ 완료된 일정의 달력 표시 유지

| 변경 전 (Before) | 변경 후 (After) |
| --- | --- |
| 일정을 완료 처리하면 달력 월간 그리드 셀 하단의 작은 점(일정 표시 점)이 즉시 사라짐 | 일정이 완료 처리되더라도 달력 날짜 셀의 일정 표시 점이 유지되어 한 달간의 전체 스케줄 흐름을 파악하기 용이하도록 개선 |

### ✨ 메모 본문 마크다운 테이블(표) 및 코드 블록 지원

| 변경 전 (Before) | 변경 후 (After) |
| --- | --- |
| 메모 본문 렌더링 시 표준 마크다운 표(`|` 문자 조합)가 일반 텍스트로 풀어져 알아보기 어려움 | 마크다운 표 표기법 및 정렬 문법(예: `:---`, `:---:`, `---:`)을 해석하여 깔끔한 HTML 표로 렌더링. 테마에 대응하는 보더와 가로 스크롤 최적화 적용 |
| 여러 줄 코드(` ``` ` 펜스 형태) 작성 시 줄바꿈이 무시되고 인라인 코드 스타일로 뭉개짐 | 펜스 코드 블록을 감지해 전용 스크롤 상자 및 우측 상단 언어 라벨 배지를 포함한 코드 블록 레이아웃으로 렌더링 |

## [v0.1.0] — 2026-05-19 · 초기 구축 (Foundation)

### ✨ 추가 (Added)

* **앱 골격**: Next.js 16(App Router) 기반 `Zero-Friction` 위젯 — 전체/일정(캘린더)/재고/메모/설정 단일 화면 전환

* **PWA**: `manifest.json`(standalone·portrait·maskable), 커스텀 서비스 워커(`worker/index.ts`)의 Web Push 수신 및 알림 클릭 시 기존 창 포커스, `useNotifications` 훅
